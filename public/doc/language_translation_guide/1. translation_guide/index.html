How to translate or update the strings

Arabic, Bengali, Chinese-simplified, Czech, Dutch, Filipino, French, German, Greek, Hebrew, Hindi, Indonesian, Italian, Japanese, Javanese, Khmer, Korean, Polish, Portuguese, Russian, Romanian, Spanish, Swedish, Thai, Turkish, Vietnamese, Urdu. Application is translated in above languages and files included.

As translation is the subject of improvement so you may find some inappropriate words in translation. It will be great if you help us to improve the translation. You can contribute the language files improvements, proper credits will be given in the application documentation. You can translate it to your language if your language is not listed above.

Note! You can edit the language file in your_project/resources/lang/ directory



Tips

Please create a backup of original language file ,it is mandatory. Additionally, it will be helpful in case of translation failure or raise an issue. Please follow these steps.

  1. Go to your_project/resources/lang/
  2. Open the file with notepad or any code editor like notepad++
  3. Do not open the file with Microsoft word like editors, they are not designed for coding.
  4. Do not use single quote (') between words and be careful about special characters.

Example: your_project/resources/lang/xx/accounts.php 


'account_type'=>'Account Type',
//Change right side string  'Account Type'

'account_types'=>'Account Types',
